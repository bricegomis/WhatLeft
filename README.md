✨ WhatLeft in a nutshell
WhatLeft is a personal planning application designed to help you decide:

what should be done now
and what can wait

The goal is not to do more, but to make better choices about how you spend your time and energy.

🧠 What WhatLeft helps you do

Manage tasks and activities with tags
Plan your days and weeks using a calendar view
Define recurring activities (sport, routines, habits)
Manage weekly quotas (gaming, series, leisure, etc.)
Take real-life constraints into account (e.g. alternating Week A / Week B)


🏗️ Architecture

Backend-first
One backend, multiple clients
Modular Monolith
Internal event-driven design
Stateless REST API

Tech stack

.NET 10 / ASP.NET Core
PostgreSQL 17 (Neon)
EF Core
JWT authentication (Auth0)
